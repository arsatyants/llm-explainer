Декодер
трансформера — шаг за шагом

GPT-стиль: вектор "cat" (d=4) проходит через все слои. Все числа реальные — вычислены вручную.

шаг 1 / 6

Вход декодера

GPT-декодер работает авторегрессионно: на шаге t входная последовательность — все токены до t. Цель — предсказать следующий.

Обучение (teacher forcing)
Вход: <BOS> "The" "cat"
Таргет: "The" "cat" "sat"
Все позиции параллельно (каузальная маска)
Инференс (авторегрессия)
<BOS> → "The"
<BOS> "The" → "cat"
<BOS> "The" "cat" → "sat"
Текущий шаг: id=101 id=1996 id=4937 → предсказываем id=3290 ("sat")
Главное отличие от энкодера: каузальная маска запрещает токену видеть будущее. Во время обучения все позиции считаются параллельно — маска имитирует авторегрессию.
шаг 2 / 6

Embedding + Positional Encoding

d_model = 4. Каждый токен → вектор из таблицы эмбеддингов → прибавляем синусоидальное позиционное кодирование.

Эмбеддинги (d=4):
PE(pos, 2i) = sin(pos / 100002i/4)    PE(pos, 2i+1) = cos(pos / 100002i/4)
Позиционное кодирование PE (фактические значения):
x = emb + PE  ↓
x = emb + PE (вход в трансформер):
"cat" (поз. 2): emb = [−0.10, +0.90, +0.30, +0.60] + PE(2) = [+0.91, −0.42, +0.02, +1.00] → x₂ = [+0.81, +0.48, +0.32, +1.60]

шаг 4 / 6

Add & LayerNorm

Residual connection + нормализация. Делается дважды в каждом блоке: после attention и после FFN.

Входной вектор "cat":
x₂ = [+0.81, +0.48, +0.32, +1.60]
attn_out = [+0.824, +0.581, +0.421, +0.588]

Сумма (residual):
x₂ + attn_out = [+1.634, +1.061, +0.741, +2.188]
LayerNorm (нормализация по четырём измерениям):
x = [+1.634, +1.061, +0.741, +2.188]
μ = (1.634 + 1.061 + 0.741 + 2.188) / 4 = 1.406
σ² = ((1.634−1.406)² + (1.061−1.406)² + (0.741−1.406)² + (2.188−1.406)²) / 4
= (0.052 + 0.119 + 0.442 + 0.611) / 4 = 0.306
σ = √0.306 = 0.553

Нормализованные значения (x − μ) / σ:
dim₀: (1.634 − 1.406) / 0.553 = +0.412
dim₁: (1.061 − 1.406) / 0.553 = −0.624
dim₂: (0.741 − 1.406) / 0.553 = −1.202
dim₃: (2.188 − 1.406) / 0.553 = +1.415
Вектор "cat" после Add&Norm:
Зачем residual? Без x₂ + attn_out информация о позиции и эмбеддинге терялась бы. Residual гарантирует, что каждый слой "добавляет", а не "заменяет". LayerNorm стабилизирует обучение: среднее → 0, дисперсия → 1.
